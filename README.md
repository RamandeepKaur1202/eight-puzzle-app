# 🧩 8 Puzzle Game

## 📖 Overview

The **8 Puzzle Mobile App** is a classic sliding puzzle game where players rearrange numbered tiles to reach the correct goal configuration.

The application is built using **React, HTML, CSS, and JavaScript**, and then converted into a native Android application using **Capacitor and Android Studio**.

The goal of the puzzle is to arrange the numbered tiles in ascending order by sliding them into the empty space.

## ✨ Features

* 🧩 Interactive **3×3 sliding puzzle board**
* 🎯 **Goal board preview** showing the target configuration
* 🔢 **Move counter** to track player moves
* ⏱ **Timer** to measure puzzle solving time
* 🔀 **Shuffle** button to generate a random puzzle
* ↩️ **Undo move** functionality
* 💡 **Hint system** with limited hints
* ⚙️ **Settings panel** for difficulty and sound control
* 🎚 **Difficulty levels** (Easy, Medium, Hard)
* 🏆 **Best move tracking**
* 🎉 **Puzzle solved popup screen**
* 🤖 Optimal solution finder using A* (A-Star) algorithm
* 📱 Packaged as a **native Android app using Capacitor**

---

## 🤖 AI Solver (A* Algorithm)

The application includes an **AI-based solver** implemented using the A* search algorithm.

The algorithm:

* Explores possible puzzle states
* Uses **Manhattan Distance heuristic**
* Finds the **optimal solution with the minimum number of moves**

This allows the app to display the **best possible solution for any puzzle configuration**.

## 📱 Build Android Application

Build the React project

```id="p55o3b"
npm run build
```

Copy build files to Android project

```id="g75fgo"
npx cap copy android
```

Sync Capacitor

```id="1e4f4y"
npx cap sync android
```

Open Android Studio

```id="tb3v82"
npx cap open android
```

Then generate the APK in Android Studio.
