## рџ¤– AI Solver (A* Algorithm)

The application includes an **AI-based solver** implemented using the A* search algorithm.

The algorithm:

* Explores possible puzzle states
* Uses **Manhattan Distance heuristic**
* Finds the **optimal solution with the minimum number of moves**

This allows the app to display the **best possible solution for any puzzle configuration**.
